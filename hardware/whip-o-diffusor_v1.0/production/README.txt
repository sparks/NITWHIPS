fabrication parameters 
-------------------------
* 2 layers FR4 0.062"
* Stacking: (TOP,BOTTOM)
* Plated-through holes
* Solder mask on both side
* White solder mask (mate finish)
* Black silkscreen on both sides (TOP and BOTTOM)
* HASL finit (RoHS)
* 3mils tolerance on drills
* 7mils spacings everywhere (checked)

CAD Package
-----------
CadSoft EAGLE 5.10 Professionnal 

Gerber files format
-------------------------------------------------
* RS-274X format (Embedded D-Codes)
* 2.4 digits english
* All files are gerber-X format, except drills (.DRILL)
* origin at 0,0

Excellon files format (*.DRILL)
------------------------------------
* Excellon format (Embedded Drill Rack)
* Dimensions 2.4 digits OMIT LEADING ZEROS


included files
---------------
*.DRILL : 	drill
*.COPPER_TOP : 	copper layer TOP
*.COPPER_BOT : 	copper layer BOTTOM
*.SILK_TOP : 	silkscreen layer TOP
*.SILK_BOT : 	silkscreen layer BOTTOM
*.MASK_TOP : 	soldermask layer TOP
*.MASK_BOT : 	soldermask layer BOTTOM
*.STENCIL_TOP : stencil layer TOP
*.STENCIL_BOT : stencil layer BOTTOM
*.mnt : 	pick and place coordinates
